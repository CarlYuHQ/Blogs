20. 有效的括号

题目描述

给定一个只包括 '('、')'、'{'、'}'、'['、']' 的字符串 s，判断字符串是否有效。

有效字符串需要满足：

  1. 左括号必须用相同类型的右括号闭合；
  2. 左括号必须按正确顺序闭合；
  3. 每个右括号都对应一个同类型左括号。

76. 最小覆盖子串

题目描述

给定两个字符串 s 和 t，返回 s 中最短的子串，使其包含 t 中所有字符（含重复字符）。若不存在则返回空字符串。

示例：

  • 输入：s = "ADOBECODEBANC", t = "ABC"
  • 输出："BANC"

思路整理

这题是典型滑动窗口，核心是：

  1. need 记录目标字符需求；
  2. window 记录当前窗口统计；
  3. 右指针扩张窗口，左指针在满足条件时尽量收缩；
  4. 维护全局最短长度和起点。

代码

虽然是困难题，但本质依然是窗口计数和双指针的基本功。

543. 二叉树的直径

题目描述

给你一棵二叉树的根节点，返回该树的直径。
直径是树中任意两个节点间最长路径的边数，这条路径不一定经过根节点。

思路整理

核心是 DFS：

  1. 对每个节点计算其左右子树最大深度；
  2. 用 left_depth + right_depth 更新全局直径；
  3. 返回该节点向上的最大深度。

代码

“遍历时维护全局最优值”是这类树题很常见的套路。

198. 打家劫舍

题目描述

给定一个非负整数数组 nums，每个元素代表每间房子的金额。
相邻房子不能同晚偷，求不触发警报情况下可偷窃的最大金额。

示例：

  • 输入：[1,2,3,1]
  • 输出：4

思路整理

这是入门 DP 经典题。
状态只有两种选择：偷当前房子 / 不偷当前房子。

  • f[i]：考虑前 i 间房子的最大金额；
  • 转移：f[i] = max(nums[i - 1] + f[i - 2], f[i - 1])。

代码

这里最容易卡住的点是 nums[i - 1] + f[i - 2]，其实它天然保证了不偷相邻房子。

279. 完全平方数

题目描述

给你一个整数 n，返回和为 n 的完全平方数的最少数量。

思路整理

可以把它看成“完全背包”：

  1. 状态：f[i][j] 表示用前 i 个平方数凑出 j 的最小数量；
  2. 转移：f[i][j] = min(f[i - 1][j], f[i][j - i * i] + 1)。

代码

55. 跳跃游戏

题目描述

给你一个非负整数数组 nums，你最初位于数组第一个下标。
数组中的每个元素代表你在该位置可以跳跃的最大长度。判断能否到达最后一个下标。

示例：

  • 输入：nums = [2,3,1,1,4]
  • 输出：true

思路整理

这题的核心是维护“当前能到达的最远位置” mx：

  1. 如果某一时刻 mx < i，说明当前位置不可达，直接返回 False；
  2. 否则持续更新 mx = max(mx, i + nums[i])；
  3. 能遍历完整个数组就返回 True。

代码

31. 下一个排列

题目描述

给你一个整数数组 nums，需要原地修改为它的下一个字典序更大的排列。
如果不存在更大的排列，则重排为最小字典序（升序）。

思路整理

标准做法分三步：

  1. 从右往左找到第一个 nums[i] < nums[i + 1] 的位置；
  2. 从右往左找到第一个大于 nums[i] 的元素 nums[j] 并交换；
  3. 将 i + 1 到末尾这段反转，得到最小后缀。

代码

153. 寻找旋转排序数组中的最小值

题目描述

给定一个原本升序、后来被旋转过若干次且元素互不相同的数组 nums，返回最小元素。
要求时间复杂度为 O(log n)。

思路整理

本质是二分查找：

  1. 比较 nums[mid] 与 nums[r]；
  2. 若 nums[mid] > nums[r]，最小值在右侧，l = mid + 1；
  3. 否则最小值在左侧（含 mid），r = mid；
  4. 循环结束时 l == r，答案是 nums[l]。

代码

这题最容易写错的是边界更新：r = mid 不能写成 mid - 1，因为 mid 本身可能就是最小值。

75. 颜色分类

题目描述

给定一个包含红色、白色和蓝色、共 n 个元素的数组 nums，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

我们使用整数 0、1 和 2 分别表示红色、白色和蓝色。必须在不使用库内置 sort 函数的情况下解决这个问题。

示例：

  • 输入：nums = [2,0,2,1,1,0]
  • 输出：[0,0,1,1,2,2]

思路整理

经典荷兰国旗问题，使用三指针：

  1. i 指向 0 区间的右边界（初始为 -1）；
  2. j 指向 2 区间的左边界（初始为 n）；
  3. k 为当前遍历指针。

遇到 0 与 i+1 交换并双指针右移；遇到 2 与 j-1 交换并左移 j；遇到 1 则 k 右移。

代码

33. 搜索旋转排序数组

题目描述

整数数组 nums 按升序排列，元素互不相同。
数组在某个下标 k 上进行了旋转，给你一个目标值 target，若存在则返回下标，否则返回 -1。要求 O(log n)。

示例：

  • 输入：nums = [4,5,6,7,0,1,2], target = 0
  • 输出：4

思路整理

本质是二分，关键是判断 target 落在哪一段有序区间：

  1. 若 nums[0] <= nums[mid]，说明左半段有序；
  2. 否则右半段有序；
  3. 根据 target 是否落在有序区间内，收缩 l 或 r。

代码

容易写错的点是 if nums[0] <= nums[mid] 里的等号不能随意省略，否则区间划分会错。

15. 三数之和

题目描述

给你一个整数数组 nums，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k，并且三数之和为 0。返回所有不重复的三元组。

示例：

  • 输入：nums = [-1,0,1,2,-1,-4]
  • 输出：[[-1,-1,2],[-1,0,1]]

思路整理

排序 + 双指针：

  1. 固定最左指针 i，在 [i+1, n-1] 上用 j、k 双指针；
  2. 和大于 0 则 k--，小于 0 则 j++；
  3. 找到一组后先记录，再跳过重复元素；
  4. 若 nums[i] > 0 可直接剪枝结束。

代码

53. 最大子数组和

题目描述

给你一个整数数组 nums，找出一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

示例：

  • 输入：nums = [-2,1,-3,4,-1,2,1,-5,4]
  • 输出：6
  • 解释：连续子数组 [4,-1,2,1] 的和最大，为 6。

思路整理

经典 Kadane 算法，维护两个量：

  1. f：以当前元素结尾的最大子数组和（f = max(f, 0) + i）；
  2. ans：全局最大子数组和。
