强化学习的最终目标是寻找最优策略。本章首先定义什么是最优策略，然后引入贝尔曼最优方程（BOE）作为求解最优策略的工具。上一章（Chapter 2）介绍了任意给定策略的贝尔曼方程；本章的 BOE 是一种特殊的贝尔曼方程，其对应的策略恰好是最优的。下一章（Chapter 4）将介绍值迭代（value iteration）算法，正是求解 BOE 的迭代算法。

一、动机示例：如何改进策略？— Section 3.1

$$q_\pi(s_1, a_1) = -1 + \gamma v_\pi(s_1) = 6.2$$

二、最优状态值与最优策略 — Section 3.2

最优策略的定义（Definition 3.1）

最优策略（Optimal policy）：策略 π∗\pi^*π∗ 是最优的，如果对所有 s∈Ss \in \mathcal{S}s∈S 和任意其他策略 π\piπ，都有：

$$v_{\pi^*}(s) \geq v_\pi(s)$$

π∗\pi^*π∗ 的状态值称为**最优状态值（optimal state values）**。

这一定义引出几个基本问题：

  • 存在性：最优策略是否存在？
  • 唯一性：最优策略是否唯一？
  • 随机性：最优策略是随机的还是确定性的？
  • 算法：如何获取最优策略和最优状态值？

三、贝尔曼最优方程（BOE）— Section 3.3

3.3.1 BOE 的表达式（Equation 3.1）

对每个 s∈Ss \in \mathcal{S}s∈S，BOE 的逐元素形式为：

$$v(s) = \max_{\pi(s) \in \Pi(s)} \sum_{a \in \mathcal{A}} \pi(a \mid s) \left[ \sum_{r \in \mathcal{R}} p(r \mid s, a) , r + \gamma \sum_{s’ \in \mathcal{S}} p(s’ \mid s, a) , v(s’) \right] = \max_{\pi(s) \in \Pi(s)} \sum_{a \in \mathcal{A}} \pi(a \mid s) , q(s, a)$$

其中 v(s),v(s′)v(s), v(s')v(s),v(s′) 是待求解的未知量，且：

$$q(s, a) = \sum_{r \in \mathcal{R}} p(r \mid s, a) , r + \gamma \sum_{s’ \in \mathcal{S}} p(s’ \mid s, a) , v(s’)$$

BOE 有两个未知变量 v(s)v(s)v(s) 和 π(a∣s)\pi(a \mid s)π(a∣s)，但可以逐一求解。

右端最大化的求解

Example 3.1：考虑方程 x=max⁡y(2x−1−y2)x = \max_y (2x - 1 - y^2)x=maxy​(2x−1−y2)。先求右端关于 yyy 的最大值（y=0y=0y=0），再解 x=2x−1x = 2x-1x=2x−1 得 x=1x=1x=1。

Example 3.2：给定 q1,q2,q3q_1, q_2, q_3q1​,q2​,q3​，在约束 c1+c2+c3=1,ci≥0c_1+c_2+c_3=1, c_i \geq 0c1​+c2​+c3​=1,ci​≥0 下最大化 ∑ciqi\sum c_i q_i∑ci​qi​。最优解是将所有权重放在最大的 qiq_iqi​ 上。

因此，BOE 右端的最大值在如下策略下取得：

$$\pi(a \mid s) = \begin{cases} 1, & a = a^* \ 0, & a \neq a^* \end{cases}$$

其中 a∗=arg⁡max⁡aq(s,a)a^* = \arg\max_a q(s, a)a∗=argmaxa​q(s,a)。即最优策略选择动作值最大的动作。

3.3.2 BOE 的矩阵-向量形式（Equation 3.2）

$$v = \max_{\pi \in \Pi} (r_\pi + \gamma P_\pi v)$$

其中 max⁡π\max_\pimaxπ​ 是逐元素的最大化操作。定义：

$$f(v) = \max_{\pi \in \Pi} (r_\pi + \gamma P_\pi v)$$

则 BOE 可简写为（Equation 3.3）：

$$v = f(v)$$

这是一个非线性方程（因为包含 max 操作）。

BOE 右端的 f(v)f(v)f(v) 是压缩映射：

$$\gamma P_{\pi_2^*}(v_1-v_2) \leq f(v_1)-f(v_2) \leq \gamma P_{\pi_1^*}(v_1-v_2)$$

定义 z=max⁡{∣γPπ2∗(v1−v2)∣,∣γPπ1∗(v1−v2)∣}z = \max\{|\gamma P_{\pi_2^*}(v_1-v_2)|, |\gamma P_{\pi_1^*}(v_1-v_2)|\}z=max{∣γPπ2∗​​(v1​−v2​)∣,∣γPπ1∗​​(v1​−v2​)∣}，利用 PπP_\piPπ​ 的随机矩阵性质（每行元素非负且和为 1），可得 ∥z∥∞≤γ∥v1−v2∥∞\|z\|_\infty \leq \gamma\|v_1-v_2\|_\infty∥z∥∞​≤γ∥v1​−v2​∥∞​。

四、从 BOE 求解最优策略 — Section 3.4

求解 v∗v^*v∗

Theorem 3.3（存在性、唯一性与算法）：BOE v=f(v)=max⁡π(rπ+γPπv)v = f(v) = \max_\pi(r_\pi + \gamma P_\pi v)v=f(v)=maxπ​(rπ​+γPπ​v) 总存在唯一解 v∗v^*v∗，可通过迭代求解：

$$v_{k+1} = f(v_k) = \max_{\pi \in \Pi} (r_\pi + \gamma P_\pi v_k), \quad k = 0, 1, 2, \ldots$$

vkv_kvk​ 以**指数速度**收敛到 v∗v^*v∗。该迭代算法就是**值迭代（value iteration）**，将在 Chapter 4 详细介绍。

求解 π∗\pi^*π∗

得到 v∗v^*v∗ 后，最优策略为：

$$\pi^* = \arg\max_{\pi \in \Pi} (r_\pi + \gamma P_\pi v^*)$$

代入 BOE 得 v∗=rπ∗+γPπ∗v∗v^* = r_{\pi^*} + \gamma P_{\pi^*} v^*v∗=rπ∗​+γPπ∗​v∗，因此 BOE 就是策略 π∗\pi^*π∗ 的贝尔曼方程。

最优性（Theorem 3.4）

v∗v^*v∗ 是最优状态值，π∗\pi^*π∗ 是最优策略。即对任意策略 π\piπ：

$$v^* = v_{\pi^*} \geq v_\pi$$

证明核心（Box 3.3）：对任意 π\piπ，有 v∗−vπ≥γPπ(v∗−vπ)≥⋯≥γnPπn(v∗−vπ)v^* - v_\pi \geq \gamma P_\pi(v^* - v_\pi) \geq \cdots \geq \gamma^n P_\pi^n(v^* - v_\pi)v∗−vπ​≥γPπ​(v∗−vπ​)≥⋯≥γnPπn​(v∗−vπ​)。由于 γ<1\gamma < 1γ<1 且 PπnP_\pi^nPπn​ 元素有界，令 n→∞n \to \inftyn→∞ 得 v∗≥vπv^* \geq v_\piv∗≥vπ​。

贪心最优策略（Theorem 3.5）

对任意 s∈Ss \in \mathcal{S}s∈S，确定性贪心策略（Equation 3.7）：

$$\pi^*(a \mid s) = \begin{cases} 1, & a = a^*(s) \ 0, & a \neq a^*(s) \end{cases}$$

是最优策略，其中：

$$a^*(s) = \arg\max_a q^*(s, a), \quad q^*(s, a) = \sum_{r \in \mathcal{R}} p(r \mid s, a) , r + \gamma \sum_{s’ \in \mathcal{S}} p(s’ \mid s, a) , v^*(s’)$$

最优策略的性质

  • 唯一性：v∗v^*v∗ 唯一，但最优策略 π∗\pi^*π∗ 不一定唯一（如 Figure 3.3 中两个不同策略都是最优的）
  • 随机性：最优策略可以是随机的也可以是确定性的，但一定存在确定性贪心最优策略

五、影响最优策略的因素 — Section 3.5

影响因素分析

从 BOE 的逐元素表达式可以看出，最优状态值和最优策略由以下参数决定：

  1. 即时奖励 rrr
  2. 折扣率 γ\gammaγ
  3. 系统模型 p(s′∣s,a),p(r∣s,a)p(s' \mid s, a), p(r \mid s, a)p(s′∣s,a),p(r∣s,a)（固定）

基线示例（Figure 3.4(a)）

rboundary=rforbidden=−1r_{\text{boundary}} = r_{\text{forbidden}} = -1rboundary​=rforbidden​=−1，rtarget=1r_{\text{target}} = 1rtarget​=1，γ=0.9\gamma = 0.9γ=0.9。

有趣的是，智能体不害怕穿过禁止区域——因为虽然经过禁止区域会获得负奖励，但由于 γ\gammaγ 较大，穿过的总奖励反而比绕路更大。说明最优策略是远视的（far-sighted）。

折扣率的影响

  • γ=0.5\gamma = 0.5γ=0.5（Figure 3.4(b)）：智能体变得短视，不敢冒险穿过禁止区域，选择绕远路到达目标
  • γ=0\gamma = 0γ=0（Figure 3.4(c)）：极度短视，每个状态只选择即时奖励最大的动作，甚至无法到达目标区域

状态值的空间分布呈现有趣的规律：离目标越近的状态值越大，越远的状态值越小——因为更长的轨迹经折扣后回报更小。

奖励值的影响

**Figure 3.4(d)**：将 rforbiddenr_{\text{forbidden}}rforbidden​ 从 −1-1−1 改为 −10-10−10，最优策略变为完全避开禁止区域。

最优策略对奖励仿射变换的不变性（Theorem 3.6）

若每个奖励 rrr 做仿射变换 r→αr+βr \to \alpha r + \betar→αr+β（α>0\alpha > 0α>0），则最优状态值变为：

$$v’ = \alpha v^* + \frac{\beta}{1-\gamma} \mathbf{1}$$

而最优策略保持不变。

证明核心（Box 3.5）：将 v′=αv∗+c1v' = \alpha v^* + c\mathbf{1}v′=αv∗+c1（c=β/(1−γ)c = \beta/(1-\gamma)c=β/(1−γ)）代入变换后的 BOE，利用 Pπ1=1P_\pi \mathbf{1} = \mathbf{1}Pπ​1=1 验证成立。由于 v′v'v′ 是 v∗v^*v∗ 的仿射变换，动作值之间的相对关系不变，贪心策略不变。

避免无意义的绕路（Figure 3.5）

每步移动的即时奖励为 rother=0r_{\text{other}} = 0rother​=0，最优策略会走弯路吗？

  • 最优策略（Figure 3.5(a)）：s2→s4s_2 \to s_4s2​→s4​，回报 = 1/(1−γ)=101/(1-\gamma) = 101/(1−γ)=10
  • 非最优策略（Figure 3.5(b)）：s2→s1→s3→s4s_2 \to s_1 \to s_3 \to s_4s2​→s1​→s3​→s4​，回报 = γ2/(1−γ)=8.1\gamma^2/(1-\gamma) = 8.1γ2/(1−γ)=8.1

结论：即使每步移动的即时奖励为零（不是惩罚），折扣率也会自动鼓励智能体尽快到达目标。轨迹越短，折扣回报越大。因此不需要额外添加负奖励来加速到达目标——那是仿射变换，不改变最优策略。

常见误解：初学者可能认为需要给每步添加负奖励（如 −1-1−1）来鼓励尽快到达目标。这是误解——对所有奖励加相同的值是仿射变换，不改变最优策略。折扣率已经自动实现了这个效果。

七、问答（Q&A）— Section 3.7

Q：最优策略的定义是什么？

A：如果某策略对所有状态的状态值都大于等于任何其他策略，则该策略是最优的。注意此定义仅适用于表格型强化学习。

Q：贝尔曼最优方程为什么重要？

A：因为它同时刻画了最优策略和最优状态值。求解该方程即可得到最优策略和最优状态值。

Q：BOE 是贝尔曼方程吗？

A：是的。BOE 是一种特殊的贝尔曼方程，其对应的策略恰好是最优的。

Q：BOE 的解是否唯一？

A：值解（最优状态值 v∗v^*v∗）唯一，但策略解（最优策略 π∗\pi^*π∗）可能不唯一。

Q：BOE 解分析的关键性质是什么？

A：右端是压缩映射，因此可以应用压缩映射定理。

Q：最优策略存在吗？

A：存在。根据 BOE 的分析，最优策略总是存在的。

Q：最优策略唯一吗？

A：不一定。可能存在多个最优策略，但它们有相同的最优状态值。

Q：最优策略是随机的还是确定性的？

A：可以是两者，但一定存在确定性的贪心最优策略。

Q：降低折扣率对最优策略有何影响？

A：最优策略变得更短视，智能体不敢冒险。

Q：对所有奖励增加相同的值，最优策略会变吗？

A：不会。这是仿射变换，最优策略不变，但最优状态值会增加。

Q：是否需要给每步添加负奖励来鼓励尽快到达目标？

A：不需要。折扣率已经自动实现了这个效果——绕路会增加轨迹长度、降低折扣回报。
