一、动机示例：均值估计的增量算法 — Section 6.1

第五章已经用 MC 估计解决了均值估计问题。现在的问题是：如何将非增量式算法转换为增量式算法？

设随机变量 XXX 的 i.i.d. 样本为 {xi}i=1n\{x_i\}_{i=1}^n{xi​}i=1n​，MC 估计为：

$$\bar{x} = \frac{1}{n} \sum_{i=1}^{n} x_i$$

非增量式方法：先收集所有样本，再计算平均值。缺点：样本数大时需要等很久。

增量式方法：定义：

$$w_{k+1} = \frac{1}{k} \sum_{i=1}^{k} x_i, \quad w_k = \frac{1}{k-1} \sum_{i=1}^{k-1} x_i$$

则可以推导出：

$$w_{k+1} = \frac{1}{k}\left((k-1)w_k + x_k\right) = w_k - \frac{1}{k}(w_k - x_k)$$

这就是增量算法：

$$w_{k+1} = w_k - \frac{1}{k}(w_k - x_k)$$

可以验证：w1=x1w_1 = x_1w1​=x1​，w2=x1w_2 = x_1w2​=x1​，w3=12(x1+x2)w_3 = \frac{1}{2}(x_1 + x_2)w3​=21​(x1​+x2​)，… ，wk+1=1k∑i=1kxiw_{k+1} = \frac{1}{k}\sum_{i=1}^k x_iwk+1​=k1​∑i=1k​xi​。

优势：每收到一个新样本，就能立即更新估计值。

进一步，考虑更一般的表达式：

$$w_{k+1} = w_k - \alpha_k(w_k - x_k)$$

当 αk=1/k\alpha_k = 1/kαk​=1/k 时回到上面的增量算法。对于一般的 αk>0\alpha_k > 0αk​>0，虽然无法得到解析表达式，但下一节将证明：如果 {αk}\{\alpha_k\}{αk​} 满足某些温和条件，wk→E[X]w_k \to \mathbb{E}[X]wk​→E[X]。

二、Robbins-Monro 算法 — Section 6.2

2.1 问题描述

随机逼近是一类用于求解求根或优化问题的随机迭代算法。与许多其他求根算法（如梯度法）不同，随机逼近不需要目标函数或其导数的表达式。

Robbins-Monro（RM）算法是随机逼近领域的开创性工作。问题设定如下：

假设要找方程 g(w)=0g(w) = 0g(w)=0 的根，其中 w∈Rw \in \mathbb{R}w∈R 是未知变量，g:R→Rg: \mathbb{R} \to \mathbb{R}g:R→R 是函数。但 ggg 的表达式未知——我们只能获得 g(w)g(w)g(w) 的带噪声观测：

$$\tilde{g}(w, \eta) = g(w) + \eta$$

其中 η\etaη 是观测误差。这是一个黑箱系统：只知道输入 www 和带噪声输出 g~\tilde{g}g~​。

2.2 RM 算法

RM 算法为：

$$w_{k+1} = w_k - a_k \tilde{g}(w_k, \eta_k), \quad k = 1, 2, 3, \ldots$$

该算法不需要函数的任何信息，只需要输入和输出。

示例：设 g(w)=w3−5g(w) = w^3 - 5g(w)=w3−5，真实根为 51/3≈1.715^{1/3} \approx 1.7151/3≈1.71。观测 g~(w)=g(w)+η\tilde{g}(w) = g(w) + \etag~​(w)=g(w)+η，η∼N(0,1)\eta \sim \mathcal{N}(0, 1)η∼N(0,1)。初始猜测 w1=0w_1 = 0w1​=0，ak=1/ka_k = 1/kak​=1/k。如 Figure 6.3 所示，尽管观测被噪声干扰，wkw_kwk​ 仍然收敛到真实根。

三、Dvoretzky 收敛定理 — Section 6.3

本节数学性较强。对收敛分析感兴趣的读者建议学习，否则可跳过。

3.1 定理内容

定理 6.2（Dvoretzky 定理）：考虑随机过程

$$\Delta_{k+1} = (1 - \alpha_k)\Delta_k + \beta_k \eta_k$$

其中 αk≥0\alpha_k \geq 0αk​≥0，βk≥0\beta_k \geq 0βk​≥0。如果以下条件满足：

(a) ∑k=1∞αk=∞\sum_{k=1}^{\infty} \alpha_k = \infty∑k=1∞​αk​=∞，∑k=1∞αk2<∞\sum_{k=1}^{\infty} \alpha_k^2 < \infty∑k=1∞​αk2​<∞，∑k=1∞βk2<∞\sum_{k=1}^{\infty} \beta_k^2 < \infty∑k=1∞​βk2​<∞ 一致几乎必然成立；

(b) E[ηk∣Hk]=0\mathbb{E}[\eta_k | \mathcal{H}_k] = 0E[ηk​∣Hk​]=0 且 E[ηk2∣Hk]≤C\mathbb{E}[\eta_k^2 | \mathcal{H}_k] \leq CE[ηk2​∣Hk​]≤C 几乎必然成立；

则 Δk\Delta_kΔk​ 几乎必然收敛到零。

$$h_{k+1} - h_k = -\alpha_k(2-\alpha_k)\Delta_k^2 + \beta_k^2 \eta_k^2 + 2(1-\alpha_k)\beta_k \eta_k \Delta_k$$

取条件期望后，第一项 ≤0\leq 0≤0（当 αk≤1\alpha_k \leq 1αk​≤1），第二项 ≤βk2C\leq \beta_k^2 C≤βk2​C，第三项 =0= 0=0（因为 E[ηk∣Hk]=0\mathbb{E}[\eta_k|\mathcal{H}_k] = 0E[ηk​∣Hk​]=0）。

因此 ∑k=1∞E[hk+1−hk∣Hk]≤∑k=1∞βk2C<∞\sum_{k=1}^{\infty} \mathbb{E}[h_{k+1} - h_k | \mathcal{H}_k] \leq \sum_{k=1}^{\infty} \beta_k^2 C < \infty∑k=1∞​E[hk+1​−hk​∣Hk​]≤∑k=1∞​βk2​C<∞。

由拟鞅收敛定理，hkh_khk​ 几乎必然收敛。进一步可证 ∑k=1∞αkΔk2\sum_{k=1}^{\infty} \alpha_k \Delta_k^2∑k=1∞​αk​Δk2​ 有界，结合 ∑αk=∞\sum \alpha_k = \infty∑αk​=∞，必有 Δk→0\Delta_k \to 0Δk​→0 几乎必然成立。

证明思路：设 g(w∗)=0g(w^*) = 0g(w∗)=0。RM 算法减去 w∗w^*w∗：

四、随机梯度下降 — Section 6.4

4.1 SGD 算法

SGD 是机器学习中广泛使用的算法，也是 RM 算法的特例。

考虑优化问题：

$$\min_w J(w) = \mathbb{E}[f(w, X)]$$

梯度下降（GD）：

$$w_{k+1} = w_k - \alpha_k \mathbb{E}[\nabla_w f(w_k, X)]$$

需要期望值 E[∇wf(wk,X)]\mathbb{E}[\nabla_w f(w_k, X)]E[∇w​f(wk​,X)]，但概率分布通常未知。

批量梯度下降（BGD）：用 nnn 个样本的梯度平均值近似：

$$w_{k+1} = w_k - \alpha_k \frac{1}{n} \sum_{i=1}^{n} \nabla_w f(w_k, x_i)$$

需要所有样本，每步计算量大。

随机梯度下降（SGD）：每次只用一个样本：

$$w_{k+1} = w_k - \alpha_k \nabla_w f(w_k, x_k)$$

SGD 用随机梯度替换了真实梯度。由于：

$$\nabla_w f(w_k, x_k) = \underbrace{\mathbb{E}[\nabla_w f(w_k, X)]}{\text{真实梯度}} + \underbrace{(\nabla_w f(w_k, x_k) - \mathbb{E}[\nabla_w f(w_k, X)])}{\eta_k}$$

SGD 相当于梯度下降加上一个零均值扰动项 ηk\eta_kηk​，直觉上不会破坏收敛性。

4.2 应用于均值估计

均值估计可以表述为优化问题：

$$\min_w J(w) = \mathbb{E}\left[\frac{1}{2}|w - X|^2\right]$$

最优解为 w∗=E[X]w^* = \mathbb{E}[X]w∗=E[X]。梯度为 ∇wf(w,X)=w−X\nabla_w f(w, X) = w - X∇w​f(w,X)=w−X。

SGD 给出：wk+1=wk−αk(wk−xk)w_{k+1} = w_k - \alpha_k(w_k - x_k)wk+1​=wk​−αk​(wk​−xk​)，正是增量均值估计算法！

4.3 SGD 的收敛模式

SGD 有一个有趣的收敛模式：

相对误差 δk=∣∇wf(wk,xk)−E[∇wf(wk,X)]∣∣E[∇wf(wk,X)]∣\delta_k = \frac{|\nabla_w f(w_k, x_k) - \mathbb{E}[\nabla_w f(w_k, X)]|}{|\mathbb{E}[\nabla_w f(w_k, X)]|}δk​=∣E[∇w​f(wk​,X)]∣∣∇w​f(wk​,xk​)−E[∇w​f(wk​,X)]∣​

可以证明：

$$\delta_k \leq \frac{\text{随机梯度与真实梯度的偏差}}{c \cdot |w_k - w^*|}$$

当 wkw_kwk​ 远离 w∗w^*w∗ 时：δk\delta_kδk​ 很小，SGD 行为类似梯度下降，快速收敛。

当 wkw_kwk​ 接近 w∗w^*w∗ 时：δk\delta_kδk​ 可能很大，收敛过程更具随机性。

Figure 6.5 展示了在 R2\mathbb{R}^2R2 上的均值估计（E[X]=0\mathbb{E}[X] = 0E[X]=0）中不同算法的表现：SGD（m=1m=1m=1）远离原点时快速接近，接近后表现出随机性；MBGD（m=50m=50m=50）收敛最快。

4.5 SGD 的收敛性

定理 6.4（SGD 收敛性）：对于 SGD 算法，如果以下条件满足：

(a) 0<c1≤∇w2f(w,X)≤c20 < c_1 \leq \nabla_w^2 f(w, X) \leq c_20<c1​≤∇w2​f(w,X)≤c2​（凸性条件）；

(b) ∑k=1∞ak=∞\sum_{k=1}^{\infty} a_k = \infty∑k=1∞​ak​=∞ 且 ∑k=1∞ak2<∞\sum_{k=1}^{\infty} a_k^2 < \infty∑k=1∞​ak2​<∞；

六、问答 — Section 6.6

Q：什么是随机逼近？
A：一类用于求解求根或优化问题的随机迭代算法。

Q：为什么需要学习随机逼近？
A：因为第七章的时序差分算法可以看作随机逼近算法。有了本章知识，理解这些算法会更自然。

Q：为什么本章频繁讨论均值估计？
A：状态值和动作值定义为随机变量的均值。第七章的时序差分算法与均值估计的随机逼近算法类似。

Q：RM 算法相比其他求根算法有什么优势？
A：不需要目标函数或其导数的表达式，是黑箱技术，只需要输入和输出。著名的 SGD 算法就是 RM 算法的特例。

Q：SGD 的基本思想是什么？
A：SGD 解决涉及随机变量的优化问题。当概率分布未知时，仅用样本即可求解。数学上，SGD 用随机梯度替换梯度下降中的真实梯度。

Q：SGD 能快速收敛吗？
A：SGD 有一个有趣的收敛模式：远离最优解时收敛快（类似梯度下降），接近最优解时随机性增大，收敛变慢。

Q：什么是 MBGD？它相比 SGD 和 BGD 有什么优势？
A：MBGD 是 SGD 和 BGD 的中间版本。相比 SGD 使用更多样本，随机性更小；相比 BGD 不需要所有样本，更灵活。
