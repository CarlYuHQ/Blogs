一、动机示例：均值估计 — Section 5.1

1.1 问题引入

均值估计是理解”从数据学习”的基础问题。为什么关心均值估计？因为状态值和动作值都定义为回报的期望值，估计状态值或动作值本质上就是均值估计问题。

考虑随机变量 XXX，任务计算其期望值 E[X]\mathbb{E}[X]E[X]。有两种方法：

方法一：有模型。 如果 XXX 的概率分布已知，可以直接按定义计算：

$$\mathbb{E}[X] = \sum_{x \in \mathcal{X}} p(x) x$$

方法二：无模型。 当概率分布未知时，假设有一些样本 {x1,x2,…,xn}\{x_1, x_2, \ldots, x_n\}{x1​,x2​,…,xn​}，可以用样本平均值近似：

$$\mathbb{E}[X] \approx \bar{x} = \frac{1}{n} \sum_{j=1}^{n} x_j$$

当 n→∞n \to \inftyn→∞ 时，xˉ→E[X]\bar{x} \to \mathbb{E}[X]xˉ→E[X]。这由大数定律保证。

1.2 示例：抛硬币

设随机变量 XXX 表示硬币正反面：X=1X = 1X=1（正面），X=−1X = -1X=−1（反面），真实分布为均匀分布 p(X=1)=p(X=−1)=0.5p(X=1) = p(X=-1) = 0.5p(X=1)=p(X=−1)=0.5。

  • 有模型：E[X]=0.5×1+0.5×(−1)=0\mathbb{E}[X] = 0.5 \times 1 + 0.5 \times (-1) = 0E[X]=0.5×1+0.5×(−1)=0
  • 无模型：抛硬币多次，计算样本平均值。如 Figure 5.2 所示，随着样本数增加，平均值逐渐收敛到真实期望值 0。

重要：用于均值估计的样本必须是独立同分布（i.i.d.）的。否则可能无法正确估计期望值。

1.3 大数定律（Box 5.1）

对随机变量 XXX，设 {xi}i=1n\{x_i\}_{i=1}^n{xi​}i=1n​ 为 i.i.d. 样本，xˉ=1n∑i=1nxi\bar{x} = \frac{1}{n}\sum_{i=1}^n x_ixˉ=n1​∑i=1n​xi​。则：

$$\mathbb{E}[\bar{x}] = \mathbb{E}[X], \quad \text{var}[\bar{x}] = \frac{1}{n}\text{var}[X]$$

这意味着 xˉ\bar{x}xˉ 是 E[X]\mathbb{E}[X]E[X] 的无偏估计，且方差随 nnn 增大趋近于零。

二、MC Basic：最简单的 MC 算法 — Section 5.2

2.1 将策略迭代转换为无模型

回顾策略迭代算法（Section 4.2），每次迭代有两个步骤：

  1. 策略评估：求解 vπk=rπk+γPπkvπkv_{\pi_k} = r_{\pi_k} + \gamma P_{\pi_k} v_{\pi_k}vπk​​=rπk​​+γPπk​​vπk​​
  2. 策略改进：求解 πk+1(s)=arg⁡max⁡π∑aπ(a∣s)qπk(s,a)\pi_{k+1}(s) = \arg\max_\pi \sum_a \pi(a|s) q_{\pi_k}(s, a)πk+1​(s)=argmaxπ​∑a​π(a∣s)qπk​​(s,a)

动作值是这两个步骤的核心。关键问题是如何计算动作值。

有模型方法（策略迭代采用）：先求解状态值，再通过模型计算动作值：

$$q_{\pi_k}(s, a) = \sum_r p(r|s,a) r + \gamma \sum_{s’} p(s’|s,a) v_{\pi_k}(s’)$$

这需要已知系统模型 {p(r∣s,a),p(s′∣s,a)}\{p(r|s,a), p(s'|s,a)\}{p(r∣s,a),p(s′∣s,a)}。

无模型方法（MC 方法采用）：动作值的定义是期望回报：

$$q_{\pi_k}(s, a) = \mathbb{E}[G_t | S_t = s, A_t = a] = \mathbb{E}[R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + \ldots | S_t = s, A_t = a]$$

由于 qπk(s,a)q_{\pi_k}(s, a)qπk​​(s,a) 是一个期望值，可以用 MC 方法估计：从 (s,a)(s, a)(s,a) 出发，按策略 πk\pi_kπk​ 与环境交互，收集 nnn 个 episode，用回报的平均值近似：

$$q_{\pi_k}(s, a) \approx \frac{1}{n} \sum_{i=1}^{n} g_{\pi_k}^{(i)}(s, a)$$

当 nnn 足够大时，根据大数定律，近似值足够准确。

核心思想：用无模型的 MC 估计替换策略迭代中有模型的策略评估步骤。注意：无模型算法直接估计动作值（而非状态值），因为从状态值计算动作值仍需要系统模型。

综合示例：Episode 长度与稀疏奖励

Figure 5.4 展示了在 5×55 \times 55×5 网格世界中（rboundary=−1r_{\text{boundary}} = -1rboundary​=−1，rforbidden=−10r_{\text{forbidden}} = -10rforbidden​=−10，rtarget=1r_{\text{target}} = 1rtarget​=1，γ=0.9\gamma = 0.9γ=0.9），不同 episode 长度下 MC Basic 的最终结果：

  • Episode 太短时（如长度 1-4），策略和值估计都不是最优的。极端情况下长度为 1 时，只有与目标相邻的状态有非零值。
  • 随着长度增加，策略和值估计逐渐趋近最优。有趣的空间模式：距离目标近的状态先获得非零值，因为从远处出发需要更多步才能到达目标并获得正奖励。
  • 当长度达到 30 时即可找到最优策略（虽然值估计还不完全最优）。

稀疏奖励问题：只有在到达目标时才能获得正奖励。这要求 episode 足够长才能到达目标，当状态空间大时学习效率低。一种解决方法是设计非稀疏奖励，在目标附近也设置小正奖励，形成”吸引场”。

三、MC Exploring Starts — Section 5.3

3.1 更高效地利用样本

MC Basic 使用初始访问（initial-visit）策略：一个 episode 仅用于估计起始状态-动作对的动作值。这不够高效，因为 episode 还访问了许多其他状态-动作对。

例如，考虑一个 episode：

$$s_1 \xrightarrow{a_2} s_2 \xrightarrow{a_4} s_1 \xrightarrow{a_2} s_2 \xrightarrow{a_3} s_5 \xrightarrow{a_1} \cdots$$

初始访问策略仅估计 (s1,a2)(s_1, a_2)(s1​,a2​) 的动作值。但可以将 episode 分解为多个子 episode：

子 episode 起始状态-动作对
原始 (s1,a2)(s_1, a_2)(s1​,a2​)
从第 2 步开始 (s2,a4)(s_2, a_4)(s2​,a4​)
从第 3 步开始 (s1,a2)(s_1, a_2)(s1​,a2​)
从第 4 步开始 (s2,a3)(s_2, a_3)(s2​,a3​)
从第 5 步开始 (s5,a1)(s_5, a_1)(s5​,a1​)

每个子 episode 的轨迹可用于估计对应状态-动作对的动作值。这样样本利用效率大大提高。

此外，同一状态-动作对可能在一个 episode 中被访问多次（如 (s1,a2)(s_1, a_2)(s1​,a2​) 被访问两次）：

  • 首次访问（first-visit）：仅统计第一次出现
  • 每次访问（every-visit）：统计每次出现

每次访问策略的样本利用效率最高。如果一个 episode 足够长，能访问所有状态-动作对多次，那么单个 episode 就可能足够估计所有动作值。

3.2 更高效地更新策略

策略一（MC Basic 采用）：收集所有从同一状态-动作对出发的 episode，用平均值估计动作值。缺点：必须等所有 episode 收集完才能更新。

策略二：用单个 episode 的回报近似动作值，立即获得粗略估计，然后逐 episode 改进策略。这属于广义策略迭代的范畴——即使值估计不够准确，也可以更新策略。

四、MC ε-Greedy：无需 Exploring Starts — Section 5.4

4.1 ε-greedy 策略

软策略（soft policy）：在任何状态都有正概率采取任何动作的策略。如果策略是软的，单个足够长的 episode 就能访问所有状态-动作对多次，从而不需要 exploring starts。

ε-greedy 策略是一种常见的软策略。设 ϵ∈[0,1]\epsilon \in [0, 1]ϵ∈[0,1]，∣A(s)∣|\mathcal{A}(s)|∣A(s)∣ 为状态 sss 的可用动作数：

$$\pi(a|s) = \begin{cases} 1 - \frac{|\mathcal{A}(s)|-1}{|\mathcal{A}(s)|}\epsilon, & \text{贪心动作} \ \frac{1}{|\mathcal{A}(s)|}\epsilon, & \text{其余 } |\mathcal{A}(s)|-1 \text{ 个动作} \end{cases}$$

  • ϵ=0\epsilon = 0ϵ=0：退化为贪心策略
  • ϵ=1\epsilon = 1ϵ=1：所有动作等概率（均匀策略）
  • 贪心动作的概率始终大于其他动作：1−∣A(s)∣−1∣A(s)∣ϵ=1−ϵ+ϵ∣A(s)∣≥ϵ∣A(s)∣1 - \frac{|\mathcal{A}(s)|-1}{|\mathcal{A}(s)|}\epsilon = 1 - \epsilon + \frac{\epsilon}{|\mathcal{A}(s)|} \geq \frac{\epsilon}{|\mathcal{A}(s)|}1−∣A(s)∣∣A(s)∣−1​ϵ=1−ϵ+∣A(s)∣ϵ​≥∣A(s)∣ϵ​

动作选择方法：生成 [0,1][0, 1][0,1] 上的随机数 xxx：

  • 若 x≥ϵx \geq \epsilonx≥ϵ：选贪心动作
  • 若 x<ϵx < \epsilonx<ϵ：在 A(s)\mathcal{A}(s)A(s) 中均匀随机选一个动作

4.3 数值示例

在 5×55 \times 55×5 网格世界中，每次迭代仅使用一个 episode（100 万步）。初始策略为均匀策略。ϵ=0.5\epsilon = 0.5ϵ=0.5 时，仅需两次迭代即可得到最优 ε-greedy 策略。

五、ε-Greedy 策略的探索与利用 — Section 5.5

探索（exploration）：策略能尽可能多地尝试不同动作，使所有动作都被充分访问和评估。

利用（exploitation）：策略应选择动作值最大的贪心动作。

ε-greedy 策略提供了平衡探索与利用的一种方式：以较高概率选择贪心动作（利用），同时以较低概率探索其他动作（探索）。

5.1 ε-greedy 策略的最优性

Figure 5.6 展示了具有相同贪心动作方向但不同 ϵ\epsilonϵ 值的 ε-greedy 策略的状态值：

  • ϵ=0\epsilon = 0ϵ=0（贪心）：状态值最大
  • 随着 ϵ\epsilonϵ 增大，状态值下降，最优性变差
  • 当 ϵ=0.5\epsilon = 0.5ϵ=0.5 时，目标状态的状态值甚至变为最小的，因为从目标出发有更高概率进入禁止区域

Figure 5.7 展示了不同 ϵ\epsilonϵ 下的最优 ε-greedy 策略：

  • ϵ=0\epsilon = 0ϵ=0：就是最优贪心策略
  • ϵ=0.1\epsilon = 0.1ϵ=0.1：最优 ε-greedy 策略与最优贪心策略一致
  • ϵ=0.2\epsilon = 0.2ϵ=0.2：不再一致。在目标状态，最优策略变成逃离（因为大概率进入禁止区域）
  • ϵ=0.5\epsilon = 0.5ϵ=0.5：偏差更大

5.2 ε-greedy 策略的探索能力

Figure 5.8 对比了不同 ϵ\epsilonϵ 的探索能力：

ϵ=1\epsilon = 1ϵ=1（均匀策略）：每个动作概率 0.2，探索能力最强。单个 episode 可以均匀访问所有状态-动作对。

**ϵ=0.5\epsilon = 0.5ϵ=0.5**：探索能力较弱。虽然也能访问所有状态-动作对，但访问次数分布极不均匀——某些动作被访问超过 25 万次，而大部分动作仅被访问几十到几百次。

实用技巧：初始设较大的 ϵ\epsilonϵ 以增强探索，然后逐渐减小以保证最终策略的最优性。

六、总结 — Section 5.6

本章介绍了第一个无模型强化学习算法。首先通过均值估计问题理解了 MC 估计的基本思想，然后介绍了三个 MC 算法：

算法 核心改进 特点
MC Basic 将策略评估从有模型改为无模型 MC 估计 最简单，揭示核心思想；样本效率低
MC Exploring Starts 使用首次/每次访问策略更好地利用样本 效率更高；需要 exploring starts 条件
MC ε-Greedy 策略改进从贪心改为 ε-greedy 去除 exploring starts 条件；平衡探索与利用

探索与利用的基本权衡：ϵ\epsilonϵ 越大，探索能力越强但最优性越差；ϵ\epsilonϵ 越小，利用越好但探索不足。

七、问答 — Section 5.7

Q：什么是蒙特卡洛估计？
A：指一类使用随机样本来解决近似问题的技术。

Q：什么是均值估计问题？
A：基于随机样本计算随机变量期望值的问题。

Q：如何解决均值估计问题？
A：两种方法：有模型（已知概率分布，按定义计算）和无模型（用 MC 估计近似）。样本数足够大时近似准确。

Q：为什么均值估计对强化学习很重要？
A：状态值和动作值都定义为回报的期望值，估计它们本质上是均值估计问题。

Q：无模型 MC 强化学习的核心思想是什么？
A：将策略迭代转换为无模型版本——用 MC 估计替换有模型的策略评估步骤。

Q：什么是初始访问、首次访问和每次访问策略？
A：不同的样本利用策略。初始访问仅用 episode 估计起始状态-动作对的值。每次访问每次出现都统计。首次访问仅统计第一次出现。

Q：什么是 exploring starts？为什么重要？
A：要求从每个状态-动作对出发都有足够多的 episode。理论上，只有每个动作值被充分探索，才能准确评估并正确选择最优动作。

Q：如何避免 exploring starts？
A：使用软策略。软策略是随机的，单个 episode 就能访问多个状态-动作对。

Q：ε-greedy 策略能是最优的吗？
A：在 Πϵ\Pi_\epsilonΠϵ​ 中可以是最优的，但在所有策略 Π\PiΠ 中通常不是。

Q：单个 episode 能访问所有状态-动作对吗？
A：可以。如果策略是软的（如 ε-greedy）且 episode 足够长。

Q：MC Basic、MC Exploring Starts 和 MC ε-Greedy 的关系？
A：MC Basic 揭示核心思想。MC Exploring Starts 通过改进样本利用效率得到。MC ε-Greedy 通过去除 exploring starts 条件得到。基本思想简单，但追求更好性能时会增加复杂性。
